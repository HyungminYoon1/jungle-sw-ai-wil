# Week 4 - 핵심 역량 목표 달성률 평가

> 📅 작성 시점: 주차 마무리 시점

한 주가 마무리되는 시점에 목표에 대한 달성율을 핵심 역량별로 기입합니다.

---

## 1. 문제해결
- **목표:** basic 6개 + 난이도 하 4개 + 난이도 중 8개 = 총 18개 과제 중 18개 해결
- **달성률:** 100% (18 / 18)
- **회고:** 이번 주에 계획한 문제를 모두 해결하며 문제 해결 목표를 달성했다.


## 2. 설계
- **목표:** 수요 코딩회 Virtual DOM + Diff 알고리즘 구현 시 DOM 구조, Virtual DOM 변환, Diff/Patch 흐름 등 설계 포인트 3개 이상을 README 또는 설계 메모에 명시
- **달성률:** 100% (3 / 3)
- **회고:** 핵심 설계 포인트를 정리하고 구현에 반영해 설계 목표를 충족했다.


## 3. 구현
- **목표:** basic 6개 테스트 통과 + 난이도 하·중 12개 AC + 수요 코딩회 핵심 기능 1개 구현 = 19개 구현 완료
- **달성률:** 100% (19 / 19)
- **회고:** 알고리즘 문제 풀이와 수요 코딩회 핵심 기능 구현까지 모두 완료했다.


## 4. 품질
- **목표:** 난이도 하·중 12개 백준 AC 통과 + 수요 코딩회 테스트 케이스 작성 및 통과 = 13개 항목
- **달성률:** 100% (13 / 13)
- **회고:** 문제 풀이 결과와 테스트 항목을 모두 통과해 품질 목표를 달성했다.


## 5. 유지보수
- **목표:** 이번 주(금~목) 7일 중 5일 이상 개인 레포지토리에 의미 있는 커밋 메시지로 커밋·푸시
- **달성률:** 100% (5 / 5)
- **회고:** 의미 있는 단위로 꾸준히 커밋하며 유지보수 습관 목표를 달성했다.


## 6. 협업
- **목표:** 팀 협업 룰/팁 합의(1개) + 수요 코딩회 팀 활동 참여(1개) + 코어타임 코딩테스트 3문제 이상 팀 풀이(1개) = 3개 완료
- **달성률:** 100% (3 / 3)
- **회고:** 팀 활동과 협업 과제를 모두 수행해 협업 목표를 달성했다.


## 7. 태도
- **목표:** 공통 6개(목표 수립, 달성률 평가, AI 원칙, 업무 검토, WIL, 팀 협업 룰) 중 6개 완료
- **달성률:** 100% (6 / 6)
- **회고:** 이번 주 공통 과제를 빠짐없이 수행하며 학습 태도를 꾸준히 유지했다.


## 8. 비즈니스 이해
- **목표:** Virtual DOM과 Diff 알고리즘이 왜 필요한지를 README에 1문장 이상 명시
- **달성률:** 100%
- **회고:** 기술이 필요한 이유를 문서에 정리하며 비즈니스 이해 목표를 달성했다.


## 9. AI 활용
- **목표:** AI 활용 원칙 수립(1개) + 수요 코딩회에서 AI 생성 코드 검증·원리 설명(발표 시 1회 이상) = 2개 완료
- **달성률:** 100% (2 / 2)
- **회고:** AI 활용 원칙을 지키면서 생성 코드의 원리와 검증 과정을 함께 점검했다.


## 10. 학습 민첩성
- **목표:** basic 6개 주제(트리, 이진 검색 트리, 그래프, BFS, DFS, 위상정렬 등)를 6개 모두 학습 후 적용
- **달성률:** 100% (6 / 6)
- **회고:** 이번 주 학습한 주제를 문제 풀이에 모두 적용하며 학습 민첩성을 높였다.
