## 1. 과제 유형별 AI 사용 범위

### 정글 입성 (미니 프로젝트)
| 구분 | 허용 |
|------|------|
| **허용** | 기획, 와이어프레임, 발표 준비, 기술 검토, 설계, 분석 |
| **미허용** | 코드 구현 (동료와의 협업·도움을 최대한 활용) |

|------|------|
| **허용** | AI를 최대한 적극 활용 |

### 그 외 과제 (공통 룰)
- 커리큘럼 난이도에 따라 팀별로 AI 사용 규칙을 정합니다.
- 개인별 난이도 차이를 고려하여 팀 내에서 사용 방식을 유연하게 조정할 수 있습니다.

**이번 주 우리 팀 선택:**

| 수준 | 허용 예시 | 유의사항 | 선택 |
|------|-----------|----------|------|
| 일부 사용 | 개념 학습, 문제·코드 이해 | 되도록 사용하지 않지만, 사용 시 개념 학습·이해에만 | □ |
| 학습 보조 | 개념 학습, 문제·코드 이해, 힌트 요청, 코드 리뷰, 오류 분석 | 코드 구현 제외하고 자유롭게 사용 | □ |
| 코드 생성 | 위 항목 + 일부 코드 구현, 과제 해답 생성 | AI 코드는 반드시 이해·검증 후 사용, 그대로 제출 금지 | □ |

## 3. 공통 원칙 (참고)

- **AI는 "정답 생성기"가 아니라 "학습 파트너"**로 사용합니다.
- 코드 결과보다 **이해·검증·개선 과정**이 더 중요합니다.
- 학습 시 ChatGPT 'Study Mode' 또는 Gemini 'Guided Learning' 옵션 활용을 권장합니다.
- 코드 구현 시에는 **최대한 AI를 사용하지 않고**, 동료와의 협업·도움을 우선합니다.
