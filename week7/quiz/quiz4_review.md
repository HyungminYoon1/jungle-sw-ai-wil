# 문제
4. 메모리에는 다음과 같은 블록들이 있으며, 각 블록의 크기는 괄호 안에 표시되어 있다.
메모리 블록: A(10), B(50), C(25), D(30), E(40)

다음 순서로 메모리 요청이 들어온다.
요청 순서: 1(30), 2(25), 3(25), 4(10)

First-fit에 대한 예시를 보고, Next-fit과 Best-fit일 때요청에 대한 메모리 할당 순서를 기록하시오. 

예) First-fit
1 - B
2 - C
3 - D
4 - A

---

# 답변
Next-fit
1 - B
2 - C
3 - D
4 - E

Best-fit
1 - D
2 - C
3 - E
4 - A


---

# 정답
Next-fit
1 - B
2 - C
3 - D
4 - E

Best-fit
1 - D
2 - C
3 - E
4 - A


// 풀이
Next-fit: 이전에 찾은 위치 다음부터 계속 찾으므로 B -> C -> D -> E
Best-fit: 그때그때 들어갈 수 있는 블록 중 가장 작은 블록을 고르므로 D -> C -> E -> A
